# 图片转 Minecraft 皮肤工具

### 参数
| 参数                                | 介绍                                                                      |
|:----------------------------------|:------------------------------------------------------------------------|
| `-i` `--input <文件名>`              | 源图片位置                                                                   |
| `-o` `--output <文件名>`             | 导出的皮肤文件位置                                                               |
| `-f` `--fit-mode <fill \| cover>` | 适应模式： <br/> `fill` - 拉伸以填充整面皮肤 <br/> `cover` - 等比放大图片以覆盖整面皮肤，但可能会裁剪部分内容 |
